🎯 CHECKLIST PÓS-DEPLOYMENT:

Imediatamente após auto-migrate.sh:
  ☐ Anotar IP da VPS que foi mostrado
  ☐ Anotar o domínio que foi configurado

Configurar DNS (Painel Hostinger):
  ☐ Domínios → Seu domínio
  ☐ DNS → Criar Registro A
  ☐ Nome: @
  ☐ Tipo: A
  ☐ Valor: [seu IP da VPS]
  ☐ Salvar

Verificar:
  ☐ nslookup seu-dominio.com (deve retornar seu IP)
  ☐ curl https://seu-dominio.com/api/health ({"status": "ok"})
  ☐ Abrir no navegador: https://seu-dominio.com

Optional:
  ☐ Configurar backup automático
  ☐ Configurar monitoramento (cron healthcheck)
  ☐ Verificar SSL auto-renewal

═══════════════════════════════════════════════════════════════════

📞 SUPORTE RÁPIDO:

Erro: "Connection refused"
  → Verifique DNS: nslookup seu-dominio.com
  → Verifique containers: docker-compose ps

Erro: "Certificate error"
  → Aguarde DNS propagar
  → Verifique SSL: ssh seu-usuario@seu-ip "ls -la /opt/ui-ux-pro-max/ssl/"

Erro: "Timeout"
  → Verifique firewall da VPS
  → Verifique logs: docker-compose logs app
